## Apa isi project ini :v?

Project ini berisi script Python yang:
- Menghasilkan file Python lain
- File hasilnya berisi **1 juta baris**
- Setiap baris adalah:
  ```python
  print("Cowok: ...")
  print("Cewek: ...")
````

* Percakapan bersifat:

  * Acak
  * Natural
  * Random
  * Kadang nggak nyambung (seperti chat asli)

---

## Cara Menggunakan

### 1. Clone repository

```bash
git clone https://github.com/username/sejuta-baris-percakapan.git
cd sejuta-baris-percakapan
```

### 2. Jalankan generator

```bash
python generate.py
```

### 3. Hasil

Akan muncul file:

```
percakapan_sejuta_baris.py
```

Isi file tersebut adalah **1.000.000 baris kode Python**.
